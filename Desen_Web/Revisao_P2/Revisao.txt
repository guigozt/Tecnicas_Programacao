QUESTÃO-01: 
    O Node.js possui uma arquitetura diferente dos servidores web tradicionais (como o
    Apache com PHP ou Java), sendo single-threaded. Explique como o Node.js consegue suportar um
    grande número de requisições concorrentes utilizando o Event Loop e as operações de I/O (Entrada
    e Saída) não-bloqueantes.
RESPOSTA:
    Diferentes de servidores tradicionais, ele não cria thread para cada requisição. Em vez disso, existe uma unica thread principal responsável por executar o codigo.
    Quando ocorre uma operação demorada , como leitura de arquivos ou acesso ao banco de dados, O node delega isso para o sistema operacional/libuv, e continua execuatando outras requisições sem bloquear a execução.
    O Event Loop monitora essas operações assincronas. Quando uma operaçõa termina, o callback é colocado em uma fila e executado posteriormente pelo thread.

QUESTÃO-04: 
    Ao se comunicar com um Web Service via HTTP, é retornado um código sinalizando o
    status da operação. Explique o que significam os códigos que iniciam com 2XX (ex: 200, 201), 4XX e
    5XX.
RESPOSTA:
    Códigos 2XX: Indicam sucesso na operação.
        - 200 -> Requisição executada com sucesso
        - 201 -> Recurso criado com sucesso
    Códigos 4XX: Indicam erro causado pelo cliente.
        - 400 -> Requisição inválida
        - 401 -> Usuario não autenticado
        - 404 -> Recurso não encontrado
    Codigos 5XX: Indicam erro no servidor
        - 500 -> Erro generico no servidor
        - 503 -> Serviço indisponivel

QUESTÃO-09: 
    Você deve criar um servidor Node.js simples do zero que comunique via protocolo
    HTTP. Usando o módulo http, escreva o código para criar um servidor que escute a porta 3000 e
    retorne uma resposta com status 200, o Content-Type como text/plain e exiba a mensagem: "Ola
    Mundo Node!\n".
RESPOSTA
    const http = require('http')
    const servidor = http.createServer((req, res) => {
        res.writeHead(200, 'Content-Type': 'text/plain')
    });
    res.end('Hello World')

    servidor.listen(3000, () => {
        console.log('Servido rodando...')
    })
